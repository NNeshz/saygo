# Módulo 10 — Phrasal Verbs Esenciales

*El inglés cotidiano real · 10 horas*


**Lógica de las partículas — no memorizar de cero**

- up = completar/terminar/aumentar: finish up, use up, give up, grow up
- out = hacia afuera/completamente: find out, run out, go out, drop out
- back = retorno/repetición: call back, come back, pay back, go back
- down = reducir/registrar: write down, calm down, cut down
- away = alejamiento/desaparición: go away, throw away, put away

**Separables vs. inseparables**

- Inseparables: look for, look after, get on, get off, run out of
- Separables: turn on/off, put on, take off, fill out, pick up, drop off, call back, hang up

**Phrasal verbs por categoría**

- Tecnología: turn on/off, log in, log out, set up, back up, run out of battery
- Ropa: put on, take off, try on, dress up
- Búsqueda: look for, find out, figure out
- Social: hang out with, catch up, call back, pick up (the phone), hang up
- Escuela/trabajo: fill out (a form), hand in, drop out, keep up with, give up, catch up on
- Transporte: get on, get off, pick up, drop off
- Relaciones: get along with, break up with, make up


## 1. ¿Para qué sirve esto?

Los phrasal verbs son verbos formados por un verbo base + una partícula (preposición o adverbio) que juntos crean un significado nuevo.
Son la base del inglés cotidiano: los usas para hablar de tu celular, la ropa, el transporte, el trabajo y las relaciones.
Sin phrasal verbs, tu inglés suena formal y robótico; con ellos, suenas natural y fluido como hablante nativo.
Este módulo te enseña la lógica detrás de las partículas para que puedas adivinar el significado de phrasal verbs nuevos.


## 2. Estructura y reglas

### (a) Lógica de las partículas

| Partícula | Significado general | Ejemplos |
|-----------|---------------------|----------|
| **up** | completar / aumentar | *finish up, fill up, back up* |
| **out** | hacia afuera / completamente | *find out, figure out, log out* |
| **back** | regresar / devolver | *call back, give back, pay back* |
| **down** | reducir / registrar / descansar | *write down, calm down, slow down* |
| **away** | desaparecer / alejarse | *throw away, run away, give away* |

### (b) Separables vs. inseparables

**Regla clave:**

- **Separable:** el objeto puede ir entre el verbo y la partícula, o después de la partícula.
  - Con sustantivo: *Turn on the TV* **o** *Turn the TV on* — ambas son correctas.
  - Con pronombre: **siempre** va en el medio → *Turn **it** on* ✓ / ~~Turn on it~~ ❌

- **Inseparable:** el objeto SIEMPRE va después de la partícula — nunca se puede separar.
  - *Look for the keys* ✓ / ~~Look the keys for~~ ❌
  - *Get on the bus* ✓ / ~~Get the bus on~~ ❌

**Separable:**
- Con sustantivo: *verb* + particle + object **o** *verb* + object + particle (ambas correctas)
- Con pronombre: el pronombre SIEMPRE va en el medio → *verb* + **pronoun** + particle

**Inseparable:**
- El objeto SIEMPRE va después de la partícula → *verb* + particle + object (nunca se separa)

### (c) Phrasal verbs por categoría

| Categoría | Phrasal Verb | Significado |
|-----------|--------------|-------------|
| **Tecnología** | turn on / turn off | encender / apagar |
| | log in / log out | iniciar sesión / cerrar sesión |
| | set up | configurar |
| | back up | hacer copia de seguridad |
| | run out of (battery) | quedarse sin (batería) |
| **Ropa** | put on | ponerse (ropa) |
| | take off | quitarse (ropa) |
| | try on | probarse (ropa) |
| | dress up | arreglarse / disfrazarse |
| **Búsqueda / Información** | look for | buscar |
| | find out | averiguar / enterarse |
| | figure out | descubrir / resolver |
| **Social / Comunicación** | hang out | pasar el rato |
| | catch up | ponerse al día |
| | call back | devolver la llamada |
| | hang up | colgar (teléfono) |
| **Escuela / Trabajo** | fill out | llenar (un formulario) |
| | hand in | entregar |
| | drop out | abandonar |
| | keep up with | mantenerse al día con |
| | give up | rendirse |
| | catch up on | ponerse al corriente en |
| **Transporte** | get on | subirse (metro, autobús) |
| | get off | bajarse (metro, autobús) |
| | pick up | recoger (a alguien) |
| | drop off | dejar (a alguien en un lugar) |
| **Relaciones** | get along with | llevarse bien con |
| | break up with | terminar con (alguien) |
| | make up | hacer las paces / maquillarse |

> ⚠️ **Errores frecuentes de esta sección:**
> - *run out of* es inseparable: ~~run battery out of~~ ❌ → *run out of battery* ✓
> - *give up* NO lleva objeto directo cuando significa "rendirse": ~~Give up it~~ ❌ → *Don't give up* ✓
> - *make up* tiene dos significados muy distintos: *maquillarse* y *hacer las paces* — el contexto lo aclara.

## 5. Tabla de errores comunes

| ❌ Error común | ✓ Forma correcta | Explicación |
|---------------|-----------------|-------------|
| ~~Look for it out~~ | **Find out** about it | *Look for* = buscar algo que no tienes; *find out* = averiguar información |
| ~~Turn on it~~ | **Turn it on** | Con pronombres, el objeto va ENTRE el verbo y la partícula |
| ~~I run out battery~~ | **I ran out of battery** | *Run out of* siempre necesita "of"; es una locución completa |
| ~~She looked after for the baby~~ | **She looked after the baby** | *Look after* es inseparable y no lleva "for" extra |
| ~~He gived up~~ | **He gave up** | *Give* es irregular: give → gave → given |
| ~~I'm boring of this — I give up it~~ | **I'm bored. I give up.** | *Give up* (rendirse) no lleva objeto; *bored* no es *boring* |
| ~~Get on of the bus here~~ | **Get off the bus here** | *Get off* = bajarse; no añadir "of" extra |
| ~~She break up him~~ | **She broke up with him** | *Break up with* requiere "with"; además es irregular: break → broke |


## 6. Comparación o contraste — Separables vs. Inseparables

### Regla del pronombre (separables)

| Situación | Ejemplo correcto | Ejemplo incorrecto |
|-----------|-----------------|-------------------|
| Sustantivo — puede ir antes o después | *Turn on the TV* / *Turn the TV on* | — |
| Pronombre — SIEMPRE en el medio | *Turn **it** on* | ~~Turn on it~~ ❌ |
| Sustantivo — puede ir antes o después | *Pick up your sister* / *Pick your sister up* | — |
| Pronombre — SIEMPRE en el medio | *Pick **her** up* | ~~Pick up her~~ ❌ |

### Cuatro pares para practicar la posición del pronombre

| Phrasal Verb | Con sustantivo | Con pronombre |
|--------------|---------------|---------------|
| **turn off** (sep.) | *Turn off the alarm.* | *Turn **it** off.* |
| **fill out** (sep.) | *Fill out the form.* | *Fill **it** out.* |
| **look for** (insep.) | *Look for the keys.* | *Look for **them**.* |
| **get along with** (insep.) | *Get along with your colleagues.* | *Get along with **them**.* |

> Notice: with inseparable phrasal verbs, even pronouns go AFTER the full phrasal verb — never in the middle.


## 7. Diálogo modelo

**Context:** Valentina calls her friend Diego. She needs a ride and has lost her student ID.


> **Valentina:** Hey, Diego! Sorry I didn't call you back sooner — I ran out of battery.
>
> **Diego:** No worries! Are you at the plaza already?
>
> **Valentina:** Yes, but I can't find my student ID. I've been looking for it everywhere.
>
> **Diego:** Did you check your jacket? Maybe you took it off when you put on your hoodie.
>
> **Valentina:** Oh! You're right — I found it. Can you pick me up in front of the Oxxo?
>
> **Diego:** Sure. But I need to drop off my brother at the metro first.
>
> **Valentina:** No problem. I'll use the time to fill out the scholarship form on my phone.
>
> **Diego:** Good idea. By the way — did you figure out how to log in to the new portal?
>
> **Valentina:** Not yet. I need to find out my student number first. I'll figure it out later.
>
> **Diego:** Don't give up — it's confusing at first. I'll be there in ten minutes!
>
> **Valentina:** Perfect. I'll hang out here. See you soon!

**Traducción:**

> **Valentina:** ¡Hola, Diego! Perdón por no devolverte la llamada antes — se me acabó la batería.
>
> **Diego:** ¡No te preocupes! ¿Ya estás en la plaza?
>
> **Valentina:** Sí, pero no encuentro mi credencial. La he buscado por todos lados.
>
> **Diego:** ¿Checaste tu chamarra? Quizás te la quitaste cuando te pusiste el hoodie.
>
> **Valentina:** ¡Ah, tienes razón — la encontré! ¿Puedes recogerme frente al Oxxo?
>
> **Diego:** Claro. Pero primero tengo que dejar a mi hermano en el metro.
>
> **Valentina:** Sin problema. Voy a usar ese tiempo para llenar el formulario de la beca en mi celular.
>
> **Diego:** Buena idea. Oye, ¿averiguaste cómo iniciar sesión en el portal nuevo?
>
> **Valentina:** Todavía no. Primero necesito averiguar mi número de estudiante. Ya lo resolveré después.
>
> **Diego:** No te rindas — es confuso al principio. ¡Llego en diez minutos!
>
> **Valentina:** Perfecto. Me quedo por aquí. ¡Nos vemos!


## 8. Texto modelo

**My Hectic Monday**

Monday started badly. My alarm didn't **turn on** — I forgot to **set it up** the night before. I jumped out of bed, **put on** the first clothes I could find, and ran out. I almost forgot to **back up** my project files, but I did it fast. On the metro I tried to **log in** to my email, but my phone had **run out of** battery. I had to **figure out** the new route because my usual line was closed. I got off two stops early and had to walk. Later, my partner called but I had to **hang up** — class was starting. I **called** her **back** at lunch. Not my best Monday, but I didn't **give up**.

*(107 words)*

**Traducción:**

El lunes empezó mal. Mi alarma no se encendió — olvidé configurarla la noche anterior. Salté de la cama, me puse la primera ropa que encontré y salí corriendo. Casi olvidé respaldar los archivos de mi proyecto, pero lo hice rápido. En el metro intenté iniciar sesión en mi correo, pero el celular se había quedado sin batería. Tuve que resolver la ruta nueva porque mi línea habitual estaba cerrada. Me bajé dos estaciones antes y tuve que caminar. Más tarde, mi pareja llamó pero tuve que colgar — comenzaba la clase. Le devolví la llamada a la hora de comer. No fue mi mejor lunes, pero no me rendí.
