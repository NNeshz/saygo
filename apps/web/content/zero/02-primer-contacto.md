# Primer Contacto

**TO BE Afirmativo**

*El primer pilar — 8 horas*

---

## Pronombres personales sujeto

Memorización absoluta — estos pronombres reemplazan al sujeto de la oración:

| Pronombre | Español | Uso |
|-----------|---------|-----|
| **I** | yo | siempre mayúscula |
| **You** | vos / tú / usted | singular y plural |
| **He** | él | persona masculina |
| **She** | ella | persona femenina |
| **It** | ello | animales, cosas, ideas |
| **We** | nosotros | incluye al hablante |
| **They** | ellos / ellas | personas o cosas |

> **It** no tiene equivalente directo en español. Cuando hablamos de una cosa, animal o idea, usamos *it*: *The cat is sleeping. **It** is tired.*

## TO BE — Presente afirmativo

| Forma completa | Contracción | Español |
|---------------|-------------|---------|
| I **am** | I**'m** | Yo soy / estoy |
| You **are** | You**'re** | Vos sos / estás |
| He **is** | He**'s** | Él es / está |
| She **is** | She**'s** | Ella es / está |
| It **is** | It**'s** | Eso es / está |
| We **are** | We**'re** | Nosotros somos / estamos |
| They **are** | They**'re** | Ellos son / están |

### Contracciones: escritura vs. spoken English

```
Formal / escrito:    I am a student.
Informal / hablado:  I'm a student.
```

> En inglés hablado, las contracciones son lo natural. Usá siempre la forma contraída al hablar. La forma completa suena formal o enfática: *"I AM your father!"*

## Presentaciones básicas

```
My name is Ana.          →  Me llamo Ana.
I'm Ana.                 →  Soy Ana.

I am from Argentina.     →  Soy de Argentina.
I am Argentinian.        →  Soy argentino/a.
```

### Diálogo modelo

```
A: Hello! My name is Carlos. What's your name?
B: Hi! I'm María. Nice to meet you.
A: Nice to meet you too. Where are you from?
B: I'm from Mexico. And you?
A: I'm from Spain.
```

### 21 – 100

A partir del 21, se combina la decena + el número:

```
21 = twenty-one     32 = thirty-two
45 = forty-five     67 = sixty-seven
99 = ninety-nine   100 = one hundred
```

| Decenas | | | | |
|---------|--|--|--|--|
| 20 twenty | 30 thirty | 40 forty | 50 fifty | 60 sixty |
| 70 seventy | 80 eighty | 90 ninety | 100 one hundred | |

> **Atención:** *forty* (40) no lleva "u" — no es *fourty*. Y *eight**een*** lleva doble "e".
