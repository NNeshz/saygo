# El Entorno y Existencia

**Plurales + Artículos + There is/are + Some/Any**

*Hablando de cosas y lugares — 10 horas*

---

## Pluralización

### Regla general: + s

```
book → books      car → cars      student → students
```

### + es (sustantivos terminados en s, sh, ch, x, o)

```
box → boxes         dish → dishes
watch → watches     bus → buses
potato → potatoes   hero → heroes
```

### Consonante + y → ies

```
baby → babies       city → cities
country → countries family → families
```

### Irregulares esenciales — memorización obligatoria

| Singular | Plural |
|----------|--------|
| man | men |
| woman | women |
| child | children |
| person | people |
| foot | feet |
| tooth | teeth |
| mouse | mice |

## Artículos

### Indefinidos: a / an

Se usan cuando mencionamos algo por **primera vez** o cuando no importa cuál específicamente.

| Artículo | Cuándo | Ejemplos |
|----------|--------|---------|
| **a** | antes de sonido consonántico | a book, a car, a **u**niversity /juː/ |
| **an** | antes de sonido vocálico | an apple, an **h**our /aʊər/ (la h es muda) |

> La regla es por **sonido**, no por letra: *a university* (suena /j/), *an hour* (la h es muda, suena /aʊ/).

### Definido: the

Se usa cuando el objeto ya es **conocido** por ambas personas, o cuando hay solo uno:

```
I have a cat. The cat is orange.   (primera vez: a cat → conocido: the cat)
The sun is very hot today.          (solo hay un sol)
Can you close the door, please?     (ambos saben de qué puerta hablamos)
```

## There is / There are — Existencia

Para decir que algo **existe** en un lugar:

| | Singular | Plural |
|--|----------|--------|
| **Afirmativo** | There **is** a book. | There **are** two books. |
| **Negativo** | There **isn't** a pen. | There **aren't** any books. |
| **Interrogativo** | **Is** there a bank? | **Are** there any students? |
| **Resp. corta** | Yes, there is. / No, there isn't. | Yes, there are. / No, there aren't. |

```
There is a supermarket near here.    →  Hay un supermercado cerca.
There are three students in the room.→  Hay tres estudiantes en el aula.
Is there a hospital?                 →  ¿Hay un hospital?
```

## Some y Any

| | Cuándo | Ejemplo |
|--|--------|---------|
| **some** | afirmativas | There are **some** chairs. / I have **some** water. |
| **any** | negativas e interrogativas | There aren't **any** books. / Are there **any** students? |

```
I have some money.            →  afirmativa ✓
I don't have any money.       →  negativa ✓
Do you have any questions?    →  interrogativa ✓
```

> **Some** también aparece en ofertas y pedidos: *Would you like **some** coffee?*

## Possessive 's — Genitivo sajón

Para indicar **posesión**, se agrega **'s** al poseedor:

```
John's car          →  el auto de John
My mother's sister  →  la hermana de mi madre
The teacher's book  →  el libro del profesor
```

### Estructura

```
Poseedor + 's + objeto poseído

Carlos's house  (la casa de Carlos)
Ana's brother   (el hermano de Ana)
```

> Cuando el sustantivo ya termina en **-s** (plural), solo se agrega el apóstrofe: *my parents' house*, *the students' books*.

### Contraste con pronombres posesivos

```
John's car  =  His car        (genitivo sajón vs. pronombre posesivo)
Ana's book  =  Her book
```
