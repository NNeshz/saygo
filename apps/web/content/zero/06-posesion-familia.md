# Posesión y Familia

**Have/Has + Have Got + Familia**

*Tener / poseer — 8 a 10 horas*

| Inglés | Español | Inglés | Español |

| mother | madre | daughter | hija |
| father | padre | son | hijo |
| parents | padres | children | hijos |

```
I have one sister and two brothers.

```

## Have / Has — Repaso y ampliación

### Afirmativo

```
I / You / We / They  →  have
He / She / It        →  has
```

### Negativo

```
I / You / We / They  →  don't have
He / She / It        →  doesn't have

I don't have any brothers.
She doesn't have a car.
```

### Interrogativo

```
Do   +  I/you/we/they  +  have  +  ...?
Does +  he/she/it      +  have  +  ...?

Do you have any siblings?    →  Yes, I do. / No, I don't.
Does he have a dog?          →  Yes, he does. / No, he doesn't.
```

## Have Got — Variante británica

*Have got* es equivalente a *have* para posesión. Es muy común en inglés británico y en conversación informal:

| Have | Have Got |
|------|----------|
| I have a car. | I**'ve got** a car. |
| She has a dog. | She**'s got** a dog. |
| Do you have a pen? | **Have** you **got** a pen? |
| I don't have time. | I **haven't got** time. |

> En este nivel es suficiente **reconocer** la forma *have got* cuando la escuchás. No es necesario producirla todavía — usá *have/has* al hablar.

## Sustantivos: contables vs. incontables

### Contables — tienen plural, se pueden contar

```
a book / two books
a friend / three friends
an apple / five apples
```

### Incontables — no tienen plural, no se cuentan directamente

```
water (not: a water / two waters ❌)
money (not: a money ❌)
time  (not: three times — en este sentido ❌)
```

> Con incontables se puede especificar la cantidad con contenedores: *a **glass** of water*, *a **piece** of advice*.

---

## Much / Many / A lot of

| | Contables | Incontables |
|--|-----------|-------------|
| **many** | many books ✓ | many water ❌ |
| **much** | much books ❌ | much water ✓ |
| **a lot of** | a lot of books ✓ | a lot of water ✓ |

### Cuándo usar cada uno

| Cuantificador | Uso principal | Ejemplo |
|--------------|---------------|---------|
| **a lot of** | afirmativas (contables e incontables) | I have a lot of friends. |
| **many** | negativas e interrogativas con contables | How many brothers do you have? |
| **much** | negativas e interrogativas con incontables | I don't have much money. |

```
I have a lot of friends.           →  afirmativa ✓
Do you have many friends?          →  interrogativa ✓
I don't have much time.            →  negativa con incontable ✓
There aren't many students today.  →  negativa con contable ✓
```
