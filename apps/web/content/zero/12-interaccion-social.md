# Interacción Social Completa

**WH- Avanzadas + Integración + Clima**

*Todas las preguntas disponibles con todos los verbos — 10 horas*

---

## WH- avanzadas y específicas

### How much / How many

| | Uso | Ejemplo |
|--|-----|---------|
| **How much** | incontables / precios | How much is it? / How much water do you drink? |
| **How many** | contables | How many books do you have? / How many sisters? |

```
How much does it cost?      →  ¿Cuánto cuesta?
How much money do you have? →  ¿Cuánto dinero tenés?
How many students are there?→  ¿Cuántos alumnos hay?
How many languages can you speak? →  ¿Cuántos idiomas hablás?
```

### Which — Elección entre opciones limitadas

```
Which one do you prefer?     →  ¿Cuál preferís?
Which car is yours?          →  ¿Cuál auto es el tuyo?
Which color do you like?     →  ¿Qué color te gusta?
```

> **What vs. Which:** *What* se usa cuando las opciones son ilimitadas. *Which* cuando las opciones son limitadas o conocidas: *What sport do you like?* (deportes en general) vs. *Which sport do you prefer, football or tennis?* (entre esas dos opciones).

### Who — Persona (sujeto u objeto)

```
Who can help me?     →  ¿Quién puede ayudarme? (sujeto)
Who did you see?     →  ¿A quién viste? (objeto)
Who is she?          →  ¿Quién es ella?
Who are they?        →  ¿Quiénes son?
```

## Preguntas con HAVE/HAS

```
How many brothers do you have?    →  ¿Cuántos hermanos tenés?
What do you have in your bag?     →  ¿Qué tenés en tu bolso?
Does she have a car?              →  ¿Tiene auto?
Has she got a laptop?             →  ¿Tiene laptop? (have got — reconocimiento)
```

## Preguntas con CAN

```
What can you do?                  →  ¿Qué podés hacer?
Who can play the piano here?      →  ¿Quién sabe tocar el piano acá?
How many languages can you speak? →  ¿Cuántos idiomas hablás?
Can you help me, please?          →  ¿Me podés ayudar?
```

## Preguntas mixtas — Presente Simple + CAN + LIKE

```
What does he like?          →  ¿Qué le gusta?
Where can I buy this?       →  ¿Dónde puedo comprar esto?
How often do you exercise?  →  ¿Con qué frecuencia hacés ejercicio?
What time does she start?   →  ¿A qué hora empieza ella?
Who do you live with?       →  ¿Con quién vivís?
```

## Let's — Sugerencias inmediatas

*Let's* (= let us) se usa para hacer **sugerencias** a un grupo que incluye al hablante:

### Afirmativo

```
Let's go!             →  ¡Vamos!
Let's eat.            →  Comamos.
Let's study together. →  Estudiemos juntos.
Let's take a break.   →  Tomemos un descanso.
```

### Negativo

```
Let's not fight.     →  No peleemos.
Let's not worry.     →  No nos preocupemos.
Let's not be late.   →  No lleguemos tarde.
```

## El clima — Weather

### Pregunta esencial

```
What's the weather like?   →  ¿Cómo está el tiempo?
What's the weather like today? →  ¿Cómo está el tiempo hoy?
```

### Vocabulario

| Inglés | Español | Inglés | Español |
|--------|---------|--------|---------|
| sunny | soleado | cloudy | nublado |
| rainy | lluvioso | windy | ventoso |
| hot | caluroso | cold | frío |
| warm | templado/cálido | cool | fresco |
| snowy | nevado | foggy | neblinoso |
| stormy | tormentoso | humid | húmedo |

### Expresiones con el clima

```
It's sunny today.          →  Hoy está soleado.
It's raining.              →  Está lloviendo.
It's very hot!             →  ¡Hace mucho calor!
It's cold and windy.       →  Hace frío y viento.
It's going to rain later.  →  Va a llover más tarde.
It was cloudy yesterday.   →  Ayer estuvo nublado.
```

> **Estructura:** Para el clima siempre se usa **It** como sujeto: *It is sunny.* / *It was cold.* / *It's going to snow.* (nunca *The weather is sunny* — aunque es correcto, no es lo más natural).

---

## Diálogo modelo

```
A: What's the weather like today?
B: It's sunny but a bit cold. Let's go for a walk!
A: Good idea. Which park do you prefer?
B: The one near the river. How many people are usually there?
A: Not many on weekdays. Let's go now!
B: Ok! Can you bring some water?
A: Sure, how much do you want?
B: Just a bottle, thanks.
```
