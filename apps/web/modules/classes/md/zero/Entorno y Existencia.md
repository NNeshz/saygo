## BLOQUE 6: Possessive 's (El Genitivo Sajón)

### 🎯 Estructura

[Poseedor] + **'s** + [Objeto]

Ejemplo: John + 's + car = John's car

### 📊 Comparación

| **Genitivo Sajón** | **Pronombre Posesivo** |

## 🎓 HOJA DE REFERENCIA RÁPIDA

### Plurales

| **Tipo**                | **Ejemplos**                                              |
| ----------------------- | --------------------------------------------------------- |
| +s                      | book → books, pen → pens                                  |
| +es                     | box → boxes, bus → buses, watch → watches                |
| Consonante + y → ies    | baby → babies, city → cities                             |
| Vocal + y → s           | boy → boys, toy → toys                                   |
| Irregulares             | man → men, woman → women, child → children               |
|                         | person → people, foot → feet, tooth → teeth, mouse → mice |

### Artículos

| **Tipo** | **Uso**                                      | **Ejemplos**                 |
| -------- | -------------------------------------------- | ---------------------------- |
| A        | Antes de sonido consonante                   | a book, a university         |
| AN       | Antes de sonido vocal                        | an apple, an hour            |
| THE      | Cuando es específico o segunda mención       | the book, the students       |

### There is / There are

| **Estructura**                  | **Ejemplo**                                  |
| --------------------------------| ---------------------------------------------|
| There is                        | There is a book                              |
| There are                       | There are books                              |
| There isn't any / There aren't any | There isn't any milk / There aren't any books |
| Is there any...? / Are there any...? | Is there any milk? / Are there any questions? |
| Yes, there is/are - No, there isn't/aren't | Yes, there is. / No, there aren't.       |

### Posesión

| **Genitivo Sajón**                 | **Equivalente con pronombre** |
| ---------------------------------- | ----------------------------- |
| John's car                         | His car                      |
| Mary's book                        | Her book                     |
| My mother's house                  | Her house                    |
